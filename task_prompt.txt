Based on the raw implementation in experiment folder. please reimplement shadow in src folder to make it highly modular, like peft library: https://github.com/huggingface/peft.

### Proposed API is as follows:

1) implicit shadow_model
```python
from transformers import AutoModelForCausalLM
from shadow_peft import get_shadow_model, ShadowConfig

model = AutoModelForCausalLM.from_pretrained('Qwen/Qwen3-0.6B')

shadow_config = ShadowConfig(
    num_shadow_layers=1,  # for shadow model intialization
    injection_hidden_size=16,
    gate_hidden_size=10,
    alpha=0.1,
    dropout=0.1,
)
model = get_shadow_model(model, shadow_config)  # will implicitly initialize shadow_model based on the backbone model
model.print_trainable_parameters()
```


2) explicit shadow_model

```python
from transformers import AutoModelForCausalLM
from shadow_peft import get_shadow_model, ShadowConfig

model = AutoModelForCausalLM.from_pretrained('Qwen/Qwen3-4B')
shadow_model = AutoModelForCausalLM.from_pretrained('Qwen/Qwen3-0.6B')

shadow_config = ShadowConfig(
    injection_hidden_size=16,
    gate_hidden_size=10,
    alpha=0.1,
    dropout=0.1
)
model = get_shadow_model(model, shadow_config, shadow_model=shadow_model)  # explicitly pass shadow_model
model.print_trainable_parameters()
```

For model saving:

```python
model.save_pretrained('/path/to/save')
```
only save the parameters of ShadowModel, InjectionModel, and ShadowUpdateModel

For model loading:

```python
from transformers import AutoModelForCausalLM
from shadow_peft import ShadowPeftModel

model = AutoModelForCausalLM.from_pretrained('Qwen/Qwen3-0.6B')
pretrained_shadow_path = '/path/to/shadow'
model = ShadowPeftModel.from_pretrained(
    model,
    pretrained_shadow_path,
    is_trainable=False
)
```
